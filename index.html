Tic Tac Toe

by: Aryan Shah

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

tic tac toe is a fun classic. It is stragetic and very easy to master.

2. What I Learned

I learned that with a replay method I would increase the efficiency of playing the game.

3. Architecture of My Game

The first class is Main this class defines the First Login Frame and the tic tac toe functionality. The second class defines the setup and replay method.

4. Technical Implementation

I am proud of the replay method. It required the Java function of if Joptionpane was the yes option. Then it would clear the board out for the users to start another game.

5. Most Challenging Part

The replay method was very difficult because it would never get called from the second class. Also plugging in names that were taken from the JTextfield component. This was difficult because I hadn't used JTextField that much. But figured this out and I am very proud.

6. Future Fixes and Updates

I could make some better designs and colors on the board to make it more interesting and fun to use.
